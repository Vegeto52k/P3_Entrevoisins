# 🤝 Entrevoisins

## 🎯 Objectif
Améliorer une application Android existante permettant aux utilisateurs d’un même quartier de s’entraider, en ajoutant de nouvelles fonctionnalités demandées par les utilisateurs.

---

## 🧠 Contexte
Projet réalisé dans un contexte professionnel simulé chez Entrevoisins.

Suite aux retours utilisateurs, une nouvelle fonctionnalité a été demandée :
- consulter les détails d’un utilisateur
- ajouter des utilisateurs en favoris

---

## 🧪 Technologies utilisées
- Java
- Android SDK
- JUnit (tests unitaires)
- Espresso (tests instrumentalisés)
- Git / GitHub

---

## ⚙️ Fonctionnalités

### Fonctionnalités existantes
- Liste des voisins
- Interaction entre utilisateurs

### Fonctionnalités implémentées
- ✅ Affichage des détails d’un utilisateur (avatar, nom)
- ✅ Navigation vers une page dédiée utilisateur
- ✅ Ajout d’un utilisateur en favoris
- ✅ Gestion de l’état des favoris

---

## 🧪 Tests

- Mise en place et validation des tests unitaires
- Mise en place et validation des tests instrumentalisés
- ✅ Tous les tests passent avec succès

---

## 🧱 Architecture

Le projet repose sur :
- Une structure Android classique (Activities / Fragments)
- Une séparation des responsabilités
- Une gestion de données utilisateurs
